# v2.0.11 Features Batch

## Summary

7 項重大更新：Settings Web Page、Config Validator、Same-Channel Multi-Bot、Quickstart Persona、Auto-General 修正、React 修正、/view Open Access。

## Key additions

1. **Settings Web Page** — 結構化 config 編輯（form + YAML 雙向同步 + validate-before-write）
2. **Config Validator** — CLI + MCP tool，驗證 fleet.yaml/classicBot.yaml 正確性
3. **Same-Channel Multi-Bot** — composite key routing、owner-wins dedup、auto migration
4. **Quickstart Persona** — 7 步加入 persona bot、CJK adapter ID
5. **Auto-General** — secondary adapter 不搶 general
6. **React** — per-adapter reactMessageStatus
7. **/view Open Access** — GET 不需 token
